供应链解决方案（Enterprise E-commerce Platform Scheme）

什么是解决方案？

根据企业的业务形态，为企业提供量身定做电商解决方案。旨在帮助企业、 集团实现采购和销售全过程的数字化管理，通过供应链全过程的电子化、无纸化和网络化，支持企业 、集团实现采购、销售活动及其客户协同的规范化、科学化和自动化，从而更好的落实采购、销售 管理要求，统一业务流程，规范采购、销售行为，集中采购、销售数据，实现更加高效的动态管控， 提高供应链的响应速度和质量

方案特点

供应链全过程数字化

从物资采购的采购计划、采购方案、采购寻源、采购过程，到产品销售活动的生产计划、 库存与订单管理，扩大到供方/经销商管理、 合同管理、合同履约和绩效评价、资金结算等后续环节，实现了采购、销售业务全链路的数据化管理。

商务协同自动化

方案覆盖从合同起草、审批、签订、履约执行和绩效评价到结算支付和合同关闭的全过程， 最大限度实现商务协同的自动化。

技术驱动的智能化

平台基于最新的互联网微服务技术架构，大量采用大数据和人工智能技术，实现了运行高效、成熟稳定和良好的可扩展性。

智慧采购方案

智慧采购系统，是中联钢在多家大、中、小型企业采购需求的分析与电商化实施的基础上， 结合企业采购流程梳理、优化再造理念开发的一体化功能与应用。对于企业、集团采购活动， 从需求形成、计划生成与审批、标书制定、公告发布、网上投标到开标、评标、监督， 以及采购活动后端的绩效分析等能效管理，实现全流程的电子化、程序化、规范化管理。 对于自采项目提供询比价、招投标、网上竞价等采购模式方便企业实现智慧化、快速高效的采购。

智慧采购——系统特色

全流程电子化管理

涵盖从需求、计划、采购到数据档案的全过程电子化采购， 是开放的、互操作的一整套应用方案，把人、过程和信息 有效地集成在一起，从而规范采购行为、提高采购效率、 降低采购成本。

全业务覆盖

系统依据《中华人民共和国招标投标法》 、《中华人民共 和国政府采购法》 、《电子招标投标系统技术规范》等相 关法律法规的要求，全面支持企业开展物资、工程、服务 等各类标的物采购，满足企业全方位电子化采购需求。

全新技术融合及安全保障

系统集成云计算、微信、移动APP、大数据等多种形式的跨 平台智能应用服务，满足各类用户多元化需求，提高工作效 率，提升用户体验，开创智能电子化采购的新时代。同时依 托中联钢先进的加密技术，采用网络级、数据库级、数据级、 各应用子系统级等多级安全保护措施，有效的保障了系统及 数据的安全。

销售电商方案

中联钢为企业提供的销售电商平台方案，是面向企业销售业务应用，以销售管理、业务为基础，以交易管理为核心，通过互联网技术串联前端客户和后端资金管理，并融合电商综合服务、销售业务绩效统计等功能，为企业提供全面的电商解决方案。方案能够充分发挥企业自主运用网络平台的能力，利用电子商务，对自有渠道和资源进行整合管理，同时加强对下属各业务部门的监督和控制。

完整交易体系

加强客户管理

以信息技术和网络技术为基础，实现企业贸易活动、内部 运营管理及供应链下游商务伙伴间协同等所有销售业务过 程的数字化、电子化。通过明晰简洁的流程设计，增强平 台客户交互体验，强化企业向心力、提升了客户管理水平、 加强客户关系管理。

销售云方案

企业销售电商平台在自成一体的同时，也能够通过中联钢 电商云平台接入流量，以客户信用体系为基础，共享钢铁 行业海量的优质客户资源，拓展营销渠道，优化流程管理， 从而提高效率、增加效益，并提升现代化管理水平。
