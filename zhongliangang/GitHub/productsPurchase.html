电子采购/ E-procurement System

系统概述

中联钢电商是行业内最早开展电子采购的平台，结合19年行业电商运营、实施经验不断完善升级的，形成现在包括计划、采购、合同、金融、供应商等流程和服务的全方位立体化新型第三方电子采购平台。业务流程涵盖了企业物资采购的所有过程，实现了采购过程的电子化管理和运作，实现了采购理念和信息技术完美结合，从而使相关人员可以轻松操作采购项目。

功能模块

  • 采购计划

    • 采购计划是采购的源头，需用部门产生物料需求，并通过相关部门审批、整合等操作形成采购计划。 对于已有ERP的企业，一般会在ERP中进行生产性物资的采购计划管理。中联钢的采购计划模块支持从ERP读取生产性物资的采购计划；对于没有ERP的企业，可在中联钢系统中创建采购申请、形成采购计划，最终编制为采购标书。

  • 招标采购

    • 包括标书制定、公告和通知、资格审查、澄清与响应、投标、开标、议标、评标、授标（决标）等，适用于复杂、价值较高、重要程度高、供应商资源丰富、监管要求高、需要通过集体综合评议确定中标商的采购品。如大宗生产物资的年度采购、设备采购、工程服务采购等。

  • 议标采购

    • 在招标采购基础上增加在线商谈程序，该系统适用于用户对招标采购的特殊要求，在提供标书发布，标书下载、网上投标、网上开标的功能基础上，提供强大的一对一、一对多的在线议标功能。议标系统符合企业电子采购的实际需要，其针对性强，灵活性大等特点已获得国内诸多应用企业的认可。

  • 询价采购

    • 简单、快捷的在线定单采购方式，适用于简单、价值相对较低、重要程度低、供应商资源丰富、监管要求低、以价格为主要决标因素的采购品。用户可以通过平台发布询价单，采、供双方通过网上洽谈确定最后定单，快捷省时且互动性强。

  • 竞价采购

    • 竞价采购又叫反拍卖。多家供应商对标的进行多轮次的报价，采购方向供应商公开报价或排名，依靠供应商之间自发的竞争达到定价目的。可帮助企业在有效控制风险的前提下实现合理的成本节约，适用于标准统一、价值相对较高、供应商资源丰富、以价格为主要决标因素的采购品。

  • 零库存管理

    • 根据采购商的要求，供应商在系统中发布库存明细，只有特定的采购商能够查询库存数据。采购商可以在网上发出送货通知，供应商接到通知后组织发货，同时通知采购商，采购商收货后发出结算通知。双方可以通过该系统有效地降低库存，提高物流效率。

  • 合同管理

    • 标书决标后，采购方可根据需求，根据决标数据和一定条款形成电子合同以数据电文形式发送给中标方，中标供应商经过相应的电子签名程序订立正式的电子合同；采购方也可采取线上+线下的形式制定合同，通过平台传输给对方从而达成订立。

  • 能效管理

    • 采购活动进行中或结束后，根据一定的评价体系和指标模型，对该活动进行计划命中率、库存资金占用、全生命周期管理、单位消耗、相关指标对标等全方位的能效评价，并形成电子数据归档、可视化。
系统特点

  • 阳光采购

    中联钢电子采购系统的流程设置完全符合国家法律法规要求，实现了招标全流程的电子化、 可视化，过程透明；可帮助监管部门及相关领导实现招标全流程监督及关键节点的管控。

  • 海量资源

    中联钢第三方电子采购平台云集首钢、鞍钢、太钢、京唐、山西焦化、阳煤丰喜等161家VIP 采购单位，囊括6.4万余家优质供应商及21万余家后备案后；采购品类涵盖原料、材料、设备、备品备件等2万多个品种，是一个经过实践检验的网上招投标平台。

  • 互联互通

    在保证安全的情况下，中联钢电子采购平台可以与企业ERP系统进行数据交换，企业只需简单 的开发就可以安全地实现ERP系统与中联钢第三方电子招投标平台的客户、交易等电商大数据的传输交互。
